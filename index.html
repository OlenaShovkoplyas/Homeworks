git init [название проекта] - Создание нового репозитория или получение его по существующему URL-адресу git clone [url-адрес] - Скачивает репозиторий вместе со всей его историей изменений git add [файл] - Добавление в индекс. Индексирует указанный файл для последующего коммита git commit -m "[сообщение с описанием]" - Фиксирует проиндексированные изменения и сохраняет их в историю версий git branch [имя ветки] - Создаёт новую ветку git remote add - Подключение к удаленному репозиторию git push [удалённый репозиторий] [ветка] - Загружает все изменения локальной ветки в удалённый репозиторий git rm [файл]- Удаляет конкретный файл из рабочей директории и индексирует его удаление
